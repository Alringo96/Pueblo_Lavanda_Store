El ecommerce obtiene y muestra la información de los productos desde una base de datos utilizando PHP. Cada producto incluye datos como nombre, descripción, precio, cantidad en stock y estado de promoción.
Además, cuenta con un panel de administración accesible mediante una ruta específica, donde es posible realizar operaciones CRUD (crear, leer, actualizar y eliminar productos) de forma dinámica a través de AJAX, sin recargar la página.
